~은 style.css를 의미, /는 style.scss가 있는 폴더를 의미, scss파일과 같은 위치에 css 폴더가 생성되고 그 css폴더 안에 컴파일된 style.css가 생긴다.

![image](https://github.com/aeiouzz/scss/assets/145514483/fccc4069-0498-4f65-8a66-0d1709746c0a)
